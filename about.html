About

Student Finance Tracker

A privacy-focused, accessible web application designed to help students manage their finances. Built entirely with vanilla HTML, CSS, and JavaScript — no external frameworks or libraries.

Key Features

  • Dashboard with real-time stats and 7-day spending chart
  • Transaction records with regex-powered live search
  • 4+ regex validations including duplicate-word detection
  • Sort by date, amount, or description
  • Inline editing and confirmed deletion
  • Monthly budget target with over/under alerts
  • Multi-currency support (USD, EUR, GBP, JPY)
  • Dark and light theme toggle
  • JSON import/export with structural validation
  • Full localStorage persistence
  • Mobile-first responsive design (360px → 768px → 1024px)
  • Keyboard-accessible with ARIA live regions
